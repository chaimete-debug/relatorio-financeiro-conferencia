Front-end financeiro para GitHub/Vercel

1. Publique o Apps Script como Web App.
2. Carregue estes ficheiros para o GitHub.
3. Importe o repositório no Vercel.
4. Abra o dashboard e cole a URL final do Apps Script (a que termina em /exec).
5. Clique em Guardar URL.

Endpoints esperados:
- ?action=resumo
- ?action=despesas
- ?action=ajustes_receita
- POST ?action=addDespesa
- POST ?action=addAjusteReceita
- POST ?action=updateAjusteReceita
